# Restaurant Reservation System

Full-stack restaurant reservation system built with Spring Boot, React, TypeScript and PostgreSQL.

## Overview

Restaurant Reservation System is a portfolio project designed to simulate a real restaurant booking workflow. Customers can check table availability and create reservations, while administrators can manage reservations, tables and booking statuses from a protected admin area.

The goal of this project is to demonstrate clean full-stack development using a REST API, relational database design, business validation, authentication and frontend-backend integration.

### Tools

- Git
- GitHub

- VS Code

## Main Features

- Check table availability
- Create restaurant reservations
- Cancel reservations
- Admin authentication
- Manage reservation statuses
- Manage restaurant tables
- Validate business rules on the backend
- REST API tested with Postman
